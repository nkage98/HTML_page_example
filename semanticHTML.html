HTML SEMÂNTICO

Este é um exemplo de página HTML semântica

    O objetivo desta página é demonstrar o uso de elementos HTML semânticos para fins de estruturação de conteúdo e acessibilidade.

Benefícios do HTML Semântico

  • Melhora a acessibilidade
  • Facilita a manutenção do código
  • Melhora o SEO
  • Proporciona uma melhor experiência ao usuário

    O objetivo é dar significado ao código utilizando os elementos disponíveis no HTML. Outras funções do HTML como os atributos possuem várias funções úteis. Dessa forma, o uso excessivo de <div>s estraga a semantica do código.

    Outra forma de organizar o HTML é utilizando comentários ou id's com bons nomes. Dessa forma mesmo com <div>s tudo fica mais claro.
